# Engagement_project
Employee Engagement Dashboard
📌 Introduction

This project is a Streamlit-based dashboard designed to analyze employee engagement and burnout risk within an organization. It helps in understanding how factors like department and overtime impact employee satisfaction and performance.

🎯 Objectives
To analyze employee engagement levels
To identify burnout risk among employees
To study the impact of overtime on engagement
To provide an interactive dashboard for data filtering and insights
⚙️ Features
📊 Interactive filters (Department & Overtime)
📈 Engagement score calculation
🔥 Burnout risk distribution visualization
⏱️ Overtime vs Engagement analysis
🧾 Real-time data display
🛠️ Technologies Used
Python
Pandas
Streamlit
